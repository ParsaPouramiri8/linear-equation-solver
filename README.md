# 🧮 Linear Equation Solver

A simple project that solves **first-degree (linear) equations** implemented in four different programming languages:

- 💙 C
- 💜 C++
- 💚 C#
- 🐍 Python

The purpose of this project is to demonstrate how the same algorithm can be implemented in different programming languages while keeping the logic simple and easy to understand.

## 🚀 How to Run

### 💙 C

Compile using **gcc**

```bash
gcc main.c -o linear_solver
```

Run

```bash
./linear_solver
```

---

### 💜 C++

Compile using **g++**

```bash
g++ main.cpp -o linear_solver
```

Run

```bash
./linear_solver
```

---

### 💚 C#

Compile

```bash
csc Program.cs
```

Run

```bash
Program.exe
```

> Or open the project in Visual Studio and press **F5**.

---

### 🐍 Python

Run

```bash
python main.py
```

or

```bash
python3 main.py
```

---

## 📖 Example

Equation:

```
2x + 4 = 0
```

Output:

```
x = -2
```
